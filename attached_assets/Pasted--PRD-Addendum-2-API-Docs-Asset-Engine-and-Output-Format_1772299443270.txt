3. The Asset Generation Engine (Column 3)
The third column (Preview) must house a robust "Asset Generation Engine" using html-to-image. The user must be able to visually tweak the preview and export it as an image or text.

A. Omni-Export Actions
Provide two distinct export buttons: "Copy Text" and "Download Image".

Copy Text: Must inject invisible zero-width spaces (\u200B) on empty line breaks to bypass LinkedIn formatting destruction.

Download Image: Triggers a native download of the styled html-to-image rendering. If it is an IG Carousel, slice the text into multiple slides and download them together in a ZIP file.

B. "Same-to-Same" UI Mockup Generator
When the user converts a post to its original format (e.g., X to X, or LinkedIn to LinkedIn), generate a pixel-perfect graphical mockup of that platform's native UI card.

Inject the live profile photo fetched from the API into this native mockup.

Wrap the native UI card in a padded background container. Provide a color picker to change this outer background color so it looks like a premium screenshot.

C. Output Formats: "Quote" and "Carousel"
Add a dedicated "Quote" format tab in the Workshop, alongside the Carousel format. For both Quotes and IG Carousels, apply the following styling controls in the third column:

Hampton Brand Default: The default styling for these generated images MUST be the Hampton podcast brand. Background: Cream/Off-White (#F5F5F0). Text: British Racing Green (#1B4332).

Dimensions Toggle: 1080x1080 (Square Quote), 1080x1350 (IG Portrait Carousel), and 1080x1920 (IG Story/Reels). The preview DOM node must strictly enforce these aspect ratios. The text must automatically scale, wrap, and center to fit the dimension perfectly.
